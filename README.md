 📱 AR Food Visual Health Assistance

An Augmented Reality (AR) based application developed using Unity and Vuforia that detects food items and displays health-related and nutritional information in real-time.

---

## 🚀 Features
- AR-based food detection
- Nutritional information overlay
- Interactive user interface
- Real-time visualization
